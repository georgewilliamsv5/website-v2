- **Framework:** React
- **Language:** TypeScript

## 🧪 Running Locally

1. **Clone the repo:**
   `git clone git@github.com:georgewilliamsv5/website-v2.git`
   `cd website-v2`

2. **Install Dependencies**
   `npm install`

3. **Start the dev server**
   `npm run dev`
